To build the VM:

1. Install [vagrant](http://vagrantup.com)
2. Run `vagrant up` (expected time: 1.5 hours)


To access the VM:

- username: vagrant
- password: vagrant
- connect with port forwarding: `ssh -Y -p2222 vagrant@127.0.0.1`


To stop the VM:

- `vagrant suspend`


To remove the VM:

- `vagrant destroy`
